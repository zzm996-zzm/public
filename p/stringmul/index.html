今天学习一下字符串相乘

题目描述

这是Leetcode的题目 43. 字符串相乘

Tag : 「字符串」、「模拟」

给定两个以字符串形式表示的非负整数 num1 和 num2，返回 num1 和 num2 的乘积，它们的乘积也表示为字符串形式。

示例 1：

输入: num1 = "2", num2 = "3"
输出: "6"

示例 2：

输入: num1 = "123", num2 = "456"
输出: "56088"

字符串相加

这道题和 415. 字符串相加 类似，首先需要我们看看字符串如何相加，题目要求不让转int去直接做，不然题目也没有什么意义了。所以先进行字符串的相加

其实思路还是很简单的，对于字符 ‘2’ 让他与 ‘0’ 进行相减的操作得到int值就是2

‘2’ -> 50

‘0’ -> 48

所以具体代码如下

func addStrings(num1 string, num2 string) string {
    add := 0
    ans := ""
    for i, j := len(num1) - 1, len(num2) - 1; i >= 0 || j >= 0 || add != 0; i, j = i - 1, j - 1 {
        var x, y int
        if i >= 0 {
            x = int(num1[i] - '0')
        }
        if j >= 0 {
            y = int(num2[j] - '0')
        }
        result := x + y + add
        ans = strconv.Itoa(result%10) + ans
        add = result / 10
    }
    return ans
}

所以可以发现只需要 字符串2中的每个单一的值与字符串1相乘，对于后续的数字末尾补0即可，对于相加的部分只需要复用字符串相加代码即可！

func multiply(num1 string, num2 string) string {
    if num1 == "0" || num2 == "0"{
        return "0"
    }
    ret := ""
    for i:=len(num2)-1;i>=0;i--{
        ans := ""
        //拼接0
        for j:=len(num2)-1;j>i;j--{
            ans += "0"
        }

        n2 := int(num2[i]-'0')
        inc := 0
        for k := len(num1) -1 ;k >= 0 || inc > 0;k--{
            n1 := 0
            if k >=  0{
                n1 = int(num1[k]-'0')
            }
            temp := n1  * n2 + inc 
            ans = strconv.Itoa(temp  % 10) + ans 
            inc = temp / 10 
        }   

       ret = addStrings(ret,ans)
    }

    return ret
}
